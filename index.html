Ejercicio 1

Listado de todos los usuarios registrados en la base de datos.

  • nombres
  • apellidos
  • correos

Solución
Código
SELECT names, lastname, email FROM `users`;

Ejercicio 2

Listado de departamentos registrados en la base de datos

  • nombre del departamento
  • nombre del país

Solución
Código
SELECT departaments.name, countries.name AS Country FROM `departaments` INNER JOIN countries ON departaments.countries_id = countries.id;

Ejercicio 3

Listado de escuelas por un país determinado.

  • nombre de la escuela
  • nombre del país

Solución
Código
SELECT school.name, countries.name AS Country FROM `school` INNER JOIN countries ON school.countries_id = countries.id;

Ejercicio 4

Listado de usuarios ordenados (de mayor edad a menor edad)

  • nombres
  • apellidos
  • fecha de nacimiento

Solución
Código
SELECT names, lastname, date_birth FROM `users` ORDER BY `users`.`date_birth` ASC;

Ejercicio 5

En que departamento hay más usuarios registrados.

  • (departamento
  • id

Solución
Código
SELECT departaments.name, departaments.id, COUNT(departaments_id) AS numero_usuarios FROM `users` INNER JOIN departaments WHERE users.departaments_id = departaments.id GROUP BY departaments_id ORDER BY numero_usuarios DESC LIMIT 1;

Ejercicio 6

Cual es el país con más usuarios registrados.

  • país
  • id

Solución
Código
SELECT countries.name, countries.id, COUNT(countries_id) AS numero_usuarios FROM `users` INNER JOIN departaments INNER JOIN countries WHERE users.departaments_id = departaments.id AND departaments.countries_id = countries.id GROUP BY countries_id ORDER BY numero_usuarios;

Ejercicio 7

Listar solo 3 departamentos.

  • nombre departamento
  • id

Solución
Código
SELECT departaments.name, departaments.id FROM `departaments` LIMIT 3;

Ejercicio 8

Listar la información de un usuario determinado.

  • nombres
  • apellidos
  • correos
  • nombre del colegio
  • nombre del departamento
  • nombre del país

Solución
Código
SELECT users.names, users.lastname, users.email, school.name AS escuela, departaments.name AS departament, countries.name AS country FROM `users` INNER JOIN school INNER JOIN departaments INNER JOIN countries WHERE users.school_id = school.id AND users.departaments_id = departaments.id AND departaments.countries_id = countries.id ORDER BY rand() LIMIT 1;

Ejercicio 10

. Cuantos usuarios viven en un departamento determinado.

  • cantidad de usuarios “numero”

Solución
Código
SELECT departaments.name , COUNT(departaments_id) AS numero_usuarios FROM `users` INNER JOIN departaments WHERE users.departaments_id = departaments.id GROUP BY departaments.id ORDER BY rand() LIMIT 1;

Listado de usuarios que tengan más de 20 años

Solución
Código

Ejercicio 13

Ejercicio 14

Listado de usuarios que no estén asociados a ningún departamento.

Solución
Código
SELECT * FROM users LEFT JOIN departaments ON departaments.id = users.departaments_id WHERE users.departaments_id IS NULL;

Ejercicio 15

Listado de usuarios que no estén asociados a ningún colegio.

Solución
Código
